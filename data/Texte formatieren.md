Es ist möglich, Texte in Freitextfeldern in sellify zu formatieren.  
Das betrifft z.B. den [Stammdaten](https://handbuch.sellify.de/allgemein/benutzeroberfl%C3%A4che/ "Begriffe aus sellify, Aufbau von sellify")\-Reiter "Bemerkung" oder den [Verknüpfungsdaten](https://handbuch.sellify.de/allgemein/benutzeroberfl%C3%A4che/ "Begriffe aus sellify, Aufbau von sellify")\-Reiter "Beschreibung" bei Erzeugen einer [neuen Aktivität](https://handbuch.sellify.de/funktionen/aktivit%C3%A4ten-in-sellify-erzeugen/ "Aktivitäten in sellify erzeugen") oder eines [neuen Dokuments](https://handbuch.sellify.de/funktionen/dokumente/ "Dokumente").  
In der oberen Leiste können Sie die verschiedenen Formatierungen per Klick anwenden.

![](https://image.jimcdn.com/app/cms/image/transf/dimension=690x10000:format=jpg/path/s42eb4d670de94a65/image/i7cb440f878d6c280/version/1614093598/image.jpg)

Wenn Sie den Mauszeigen auf die Symbole legen, ohne diese anzuklicken, wird Ihnen die jeweilige Funktion eingeblendet.

Mit den Pfeilen ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild1.png?version=1&modificationDate=1603725078957&cacheVersion=1&api=v2&width=49&height=21) können Sie die letzte Aktion rückgängig machen oder wiederherstellen.

Per Klick aus die Schere ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild2.png?version=1&modificationDate=1603725078927&cacheVersion=1&api=v2&width=28&height=22) können Sie markierte Abschnitte ausschneiden, sie werden dann aus dem Text entfernt und sind in der Zwischenablage gespeichert, sodass sie bei Bedarf an anderer Stelle eingefügt werden können.

Über das ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild3.png?version=1&modificationDate=1603725078910&cacheVersion=1&api=v2&width=21&height=21) Icon können Sie markierte Abschnitte kopieren; sie bleiben dann im Text stehen und sind in der Zwischenablage gespeichert, sodass sie an anderer Stelle zusätzlich eingefügt werden können.

Ausgeschnittene oder kopierte Abschnitten können Sie per Klick auf ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild4.png?version=1&modificationDate=1603725078893&cacheVersion=1&api=v2&width=23&height=22)einfügen.

Die Textgröße können Sie ändern, indem Sie auf das ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild6.png?version=1&modificationDate=1603725078847&cacheVersion=1&api=v2&width=36&height=22) Icon klicken - Sie haben die Wahl zwischen "Klein", "Standard", "Groß" und "Sehr groß".

Über ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild7.png?version=1&modificationDate=1603725078750&cacheVersion=1&api=v2&width=68&height=16)können Sie den Text "fett", "kursiv" oder "unterstrichen" formatieren.

Die Textfarbe kann ebenfalls geändert werden, indem Sie auf den kleinen Pfeil neben dem "A" klicken - dabei haben Sie die Wahl zwischen folgenden Farben:

![](https://image.jimcdn.com/app/cms/image/transf/none/path/s42eb4d670de94a65/image/i687b751471c26c05/version/1609938310/image.png)

Um alle Formatierungen in einem ausgewählten Bereich zu entfernen, markieren Sie den gewünschten Bereich und klicken Sie auf ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild9.png?version=1&modificationDate=1603725078677&cacheVersion=1&api=v2&width=26&height=21) 

Sie können den Text entweder linksbündig, rechtsbündig, zentriert oder als Blocksatz ausrichten, dafür ist das gewünschte Icon aus folgenden anzuklicken: ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild10.png?version=1&modificationDate=1603725078663&cacheVersion=1&api=v2&width=97&height=17) 

Über ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild11.png?version=1&modificationDate=1603725078650&cacheVersion=1&api=v2&width=20&height=18) können Sie eine nummerierte Liste beginnen, über ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild12.png?version=1&modificationDate=1603725078637&cacheVersion=1&api=v2&width=22&height=17) eine Aufzählungsliste. Bei erneuten Klick wird die entsprechende Listenart wieder entfernt.

Außerdem können Sie eine Tabelle per Klick auf ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild13.png?version=1&modificationDate=1603725078593&cacheVersion=1&api=v2&width=23&height=19) einfügen. Anschließend erscheint ein Pop-Up-Fenster, in welchem Sie die Größe der Tabelle (Zeilen- und Spaltenanzahl) und weitere detailliertere Einstellungen vornehmen können.

![](https://image.jimcdn.com/app/cms/image/transf/dimension=235x10000:format=png/path/s42eb4d670de94a65/image/i3521c992ce4178bc/version/1609938475/image.png)

Befindet sich der Cursor innerhalb der Tabelle, können Sie zudem auch die folgenden Icons anklicken:  
![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild15.png?version=1&modificationDate=1603725078493&cacheVersion=1&api=v2&width=209&height=19)  
Hier können Sie eine Spalte links oder rechts von der, in welcher sich der Cusor befindet, einfügen oder eine Zeile darüber oder darunter.  
Außerdem kann die ganze Tabelle über ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild16.png?version=1&modificationDate=1603725078443&cacheVersion=1&api=v2&width=24&height=20) entfernt werden oder über ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild17.png?version=1&modificationDate=1603725078417&cacheVersion=1&api=v2&width=22&height=19) die Spalte bzw. über ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild18.png?version=1&modificationDate=1603725078397&cacheVersion=1&api=v2&width=20&height=18) die Zeile.

Sie können auch Links in die Freitextfelder einfügen. Wenn Sie einen Link in die Zwischenablage kopiert haben, kann dieser über "Strg + V" oder per Klick auf ![](https://businessactswiki.atlassian.net/wiki/download/thumbnails/229474610/Bild22.png?version=1&modificationDate=1603725078127&cacheVersion=1&api=v2&width=19&height=21) im Textfeld eingefügt werden. Anschließend öffnet sich ein Pop-Up, in welchem Sie den Hyperlink bearbeiten oder den Text, welcher den Link enthält, anpassen können.

![](https://image.jimcdn.com/app/cms/image/transf/dimension=378x10000:format=png/path/s42eb4d670de94a65/image/i8ab28865542e1850/version/1609939087/image.png)

Es kann beides als Hyperlink ausgegeben werden oder Sie passen den Text unter "Anzeige" an, z.B. wäre hier das Wort "sellify" mit dem Link versehen:

![](https://image.jimcdn.com/app/cms/image/transf/dimension=378x10000:format=png/path/s42eb4d670de94a65/image/i8b931430bb12b5e8/version/1609939163/image.png)
